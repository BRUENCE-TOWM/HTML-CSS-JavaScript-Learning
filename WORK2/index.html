本页面采用页首（header）、内容（main）、页脚（footer）的纵向分层，其中内容区域又进一步采用侧边栏（sidebar）+ 介绍(content-area)的横向分栏布局。

页首（header）布局

页首布局核心CSS代码

用display: flex 实现水平布局，justify-content: space-between 让品牌、导航、功能区在水平方向均匀分布，通过align-items: center 保证垂直居中。

内容区（Main）布局

内容区核心CSS代码

使用display: flex 实现横向分栏，侧边栏通过 width: 260px 和 flex-shrink: 0 固定宽度且不缩放，介绍通过 flex-grow: 1 自动填充剩余空间，gap: 30px 控制两者间距。同时使用position: sticky;top: 30px;align-self: flex-start; 让侧边栏可以在 滚动页面时跟着滚动

侧边栏内部模块布局

“本周热门”模块核心CSS代码

用display: flex 让排名数字、海报、标题在同一行对齐，通过flex-shrink: 0 保证海报尺寸固定，还用flex-grow: 1 让标题自适应剩余宽度并通过text-overflow:ellipsis实现文字截断。

页脚（Footer）布局

页脚布局核心CSS代码

用flex-direction: column 实现标题与链接列表的纵向分布，用flex-wrap: wrap 实现自适应换行；并且使用text-align: center 实现文字居中。
